How To Play

짧은 플레이 방법

단어 3개를 입력하고 악몽을 생성하세요. 매 턴 세 선택지 중 하나를 고르면 Fear와 Sanity가 변합니다. Sanity가 낮아질수록 선택지 문구는 믿기 어려워지고, Fear가 100이 되면 악몽의 일부가 됩니다. 5턴을 살아남으면 조건에 따라 다른 엔딩이 열립니다.

Artist Statement

아티스트 스테이트먼트

이 작품은 사용자가 직접 입력한 평범한 단어가 어떻게 낯선 공포의 구조로 바뀌는지 탐구하는 브라우저 기반 웹 게임입니다. 실제 AI API 대신 단어의 성격을 분류하고 사건, 장소, 위협, 선택 결과를 조합하는 생성 규칙을 사용했습니다. 플레이어는 자신이 제공한 소재로 만들어진 악몽을 읽고 선택하지만, 정신력이 떨어질수록 인터페이스 자체를 의심하게 됩니다. 이 게임에서 탈출은 정답을 고르는 일이 아니라, 무너지는 의미 속에서 아직 자신이 결정하고 있다고 믿는 행위입니다.
